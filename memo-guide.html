📝 備忘録の書き方

このページでは、備忘録サイトの構成や編集方法をまとめています。

📌 使用しているツール

  • 💎 Ruby - Jekyllの動作に必要
  • 📄 Jekyll - 静的サイトジェネレータ
  • 🛠 Git - バージョン管理ツール
  • 🌍 GitHub Pages - サイト公開サービス

🖥 ホーム画面を編集したい場合

以下のファイルを編集してください：

  • 📌 index.html
  • 📌 _layouts/home.html

🔗 リンク先を編集したい場合

各ページのファイルを編集できます：

  • 🛠 Gitについて：git-guide.html
  • ⚡ Angularについて：angular-guide.html
  • 📝 備忘録の書き方：memo-guide.html

🚀 ローカルで備忘録を確認したい場合

  1. 💻 ローカルサーバーを起動： 
    bundle exec jekyll serve
  2. 🌐 ローカルでサイトを表示： 
    http://127.0.0.1:4000/

🛠 ローカルからGitHubに反映させる手順

変更したファイルをGitHubリポジトリにアップロードする流れを説明します。

  1. 🔍 変更されたファイルを確認： 
    git status
  2. 📌 変更をステージング： 
    git add .
  3. 💾 変更をコミット： 
    git commit -m "コミットコメント"
  4. 🌍 GitHubにPush： 
    git push origin main
