🛠 Angular CLI: コンポーネントの作成 (ng generate component)

このページでは、Angular CLI を使用してコンポーネントを作成する方法を説明します。

📌 ng generate component とは？

Angular CLI の便利なコマンドで、コンポーネントに必要なファイル群を簡単に生成できます。

🚀 コマンドの実行方法

  1. 🌐 プロジェクトのルートディレクトリでターミナルを開き、以下を実行： 
    ng generate component <component-name>
  2. ⏩ 短縮形でも利用可能： 
    ng g c <component-name>

📄 生成されるファイル

  • 📝 my-new-feature.component.ts: ロジックの記述
  • 📄 my-new-feature.component.html: テンプレートの記述
  • 🎨 my-new-feature.component.css: スタイルの記述
  • ✅ my-new-feature.component.spec.ts: テストファイル

🔧 便利なオプション

  • 📌 --module <module-name>: 特定の NgModule に自動登録
  • ✨ --standalone: スタンドアロンコンポーネントとして生成
  • 📂 --flat: フォルダなしで直接生成
  • 📍 --path <path>: 生成する場所を指定

📌 まとめ

ng generate component コマンドを使えば、開発効率が大幅に向上します。ぜひ試してみてください！
